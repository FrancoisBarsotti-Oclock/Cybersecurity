# 🏆 Challenge B101: Installation d'Hyperviseurs types 1 et 2, imbrication.
### François BARSOTTI
## 🎯 Pitch et Contexte du challenge

>* Installer et configurer un hyperviseur Type 2 (comme VMWare Workstation) sur une machine locale
>   * Installer et configurer un hyperviseur Type 1 (Proxmox) sur ce Type 2.
>   * Installer un autre Hyperviseur de Type 1 (Proxmox).
>* Au final vous devez avoir :
>   * 2 Proxmox prêts a être configurés pour la suite, et les 2 doivent se joindre (un petit ping à faire).
