# Session B302 : Zabbix installation
_« Un outil pour les superviser tous »_

### Notions du jour :

* Présentation de Zabbix
    * Zabbix est une solution open-source de supervision et de monitoring.
    * Il permet de surveiller les serveurs, équipements réseau, bases de données, applications.
    * Architecture basée sur un serveur central, des agents et des interfaces Web.
* Composants de Zabbix
    * Zabbix Server : Collecte et stocke les données de supervision.
    * Zabbix Agent : Installé sur les machines supervisées pour remonter les métriques.
    * Zabbix Web Interface : Interface graphique pour la gestion et l’analyse des données.
    * Base de données : MySQL, PostgreSQL ou MariaDB pour stocker les données
